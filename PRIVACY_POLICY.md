# Privacy Policy for Amazon Scout

**Effective date: July 14, 2026**

Amazon Scout helps users organize Amazon product listings into shopping sessions, add notes, and mark products as considering, skip, or undecided.

## Data handled by the extension

Amazon Scout stores the following information only when a user chooses to track a product:

- Product ASIN, title, and Amazon URL
- The user's shopping-session name
- The user's product status and notes
- The date and time the product was tracked

## How data is used

This information is used solely to provide the extension's product-tracking, annotation, session-management, and CSV-export features.

## Storage and sharing

All extension data is stored locally in the user's browser using Chrome's local extension storage. Amazon Scout does not operate a server, transmit data to the developer or third parties, sell user data, or use user data for advertising, profiling, creditworthiness, or lending decisions.

## Data retention and deletion

Data remains in local browser storage until the user removes individual tracked products, clears extension data through Chrome, or uninstalls the extension. Users can delete tracked products directly from the extension.

## Remote code

Amazon Scout does not use remotely hosted code. All JavaScript and other executable code is included in the extension package.

## Changes to this policy

If this policy changes, the updated version will be published at the privacy-policy URL associated with the extension listing.
